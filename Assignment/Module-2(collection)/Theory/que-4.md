##### Common list operations: concatenation, repetition, membership

**1.Concatenation**
Combines two or more lists into one.

Example:
```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]
result = list1 + list2
print(result)
```
**2.Repetition**
Repeats the elements of a list multiple times.

Example:
```python
list1 = [1, 2, 3]
result = list1 * 3
print(result)
```

**3.Membership**
Checks if an element exists in a list.

Example:
```python
list1 = [1, 2, 3, 4, 5]
if 2 in list1:
    print("yes")
else:
    print("no")
```
